# 函数

## 文本处理函数

``` sql
-- 取从左起长度为3的子串
LEFT('String', 3)

-- 取从右起长度为3的子串
RIGHT('String', 3)

-- 取字符串长度
LENGTH('String')

-- 取小写字符串
LOWER('String')

-- 取大写字符串
UPPER('String')

-- 去除字符串左边空格
LTRIM('String')

-- 去除字符串右边空格
RTRIM('String')

-- 取从左起第2到4的子串
SUBSTRING('String', 2, 4)

-- 连接字符串
CONCAT('String1', 'String2', 'String3')
```

## 时间日期处理函数

``` sql
-- UNIX秒数
UNIX_TIMESTAMP()

-- 格式化UNIX秒数
FROM_UNIXTIME()

-- 静态获取当前日期时间
NOW()

-- 动态获取当前日期时间
SYSDATE()

-- 获取当前日期
CURDATE()

-- 获取当前时间
CURTIME()

-- 获取日期
DATE('2020-07-05 21:07:03.001')

-- 获取时间
TIME('2020-07-05 21:07:03.001')

-- 获取年
YEAR('2020-07-05 21:07:03.001')

-- 获取月
MONTH('2020-07-05 21:07:03.001')

-- 获取日
DAY('2020-07-05 21:07:03.001')

-- 获取小时
HOUR('2020-07-05 21:07:03.001')

-- 获取分钟
MINUTE('2020-07-05 21:07:03.001')

-- 获取秒
SECOND('2020-07-05 21:07:03.001')

-- 获取微秒
MICROSECOND('2020-07-05 21:07:03.001')

-- 获取季度
QUARTER('2020-07-05 21:07:03.001')

-- 获取周数
WEEK('2020-07-05 21:07:03.001')

-- 获取一年中的第几天
DAYOFYEAR('2020-07-05 21:07:03.001')

-- 获取一月中的第几天
DAYOFMONTH('2020-07-05 21:07:03.001')

-- 获取一周中第几天
DAYOFWEEK('2020-07-05 21:07:03.001')

-- 获取月份英文名
MONTHNAME('2020-07-05 21:07:03.001')

-- 获取星期英文名
DAYNAME('2020-07-05 21:07:03.001')

-- 获取日期之差，返回天数
DATEDIFF('2020-08-05 21:07:03.001', '2020-07-05 21:07:03.001')

-- 获取时间之差，返回时间
TIMEDIFF('2020-08-05 21:07:03.001', '2020-07-05 21:07:03.001')
```

## 提取时间

``` sql
-- 标识符
    -- 年
    YEAR

    -- 月
    MOUTH

    -- 日
    DAY

    -- 时
    HOUR

    -- 分
    MINUTE

    -- 秒
    SECOND

    -- 微秒
    MICROSECOND

    -- 季度
    QUARTER

    -- 周
    WEEK

-- 如下
-- 获取单个
EXTRACT(Sign FROM '2020-07-05 21:07:03.001')

/*
输出
=> SIGN
*/

-- 获取组合
EXTRACT(Sign1_Sign2 FROM '2020-07-05 21:07:03.001')

/*
输出
=> Sign1Sign2
*/

-- 获取增加之后的时间
DATE_ADD('2020-07-05 21:07:03.001', INTERVAL Number Sign)

-- 获取减少之后的时间
DATE_SUB('2020-07-05 21:07:03.001', INTERVAL Number Sign)
```
